📖 What's Inside: Module-by-Module Breakdown

🔢 Module 1: Algebra Foundations

Master the basics: variables, expressions, order of operations, and real number properties. Interactive examples make abstract concepts concrete. This module builds the foundation everything else builds on.

⭐ Key Takeaway: Build strong fundamentals

➕ Module 2: Linear Equations & Inequalities

Solve one-step to multi-step equations, graph linear functions, and tackle inequalities with confidence. Real-world applications show students why this matters beyond the classroom.

📈 Key Takeaway: Solve any linear problem

📊 Module 3: Systems of Equations

Master substitution, elimination, and graphing methods. Learn when to use each strategy and apply to word problems. Students learn to connect multiple equations to solve complex problems.

🔗 Key Takeaway: Connect multiple equations

🔀 Module 4: Polynomials & Factoring

Add, subtract, multiply polynomials. Master factoring techniques: GCF, trinomials, difference of squares, and grouping. This is often the hardest module for students, but AlgePrime's step-by-step approach makes it manageable.

✂️ Key Takeaway: Break down complex expressions

📐 Module 5: Quadratic Equations

Solve quadratics by factoring, completing the square, and quadratic formula. Graph parabolas and interpret real-world applications. Students master the parabola and understand its real-world significance.

🎯 Key Takeaway: Master the parabola

📉 Module 6: Exponents & Radicals

Simplify exponential expressions, work with radicals, and solve radical equations. Includes scientific notation applications. Students learn to handle powers and roots with confidence.

🔢 Key Takeaway: Handle powers & roots

📈 Module 7: Functions & Graphing

Understand function notation, domain/range, and transform graphs. Learn to interpret and create function models. Students visualize mathematical relationships and understand how functions work in real life.

📊 Key Takeaway: Visualize mathematical relationships

🎓 Module 8: Test Prep & Advanced Topics

Prepare for standardized tests (SAT, ACT, state exams). Review key concepts and tackle challenging word problems. This module prepares students for success on standardized tests and prepares them for higher math.

🏆 Key Takeaway: Ace your exams
