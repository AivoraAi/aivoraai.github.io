Use This Exact AI Prompt Template

Copy this prompt and customize for your needs:

# Bulk Meta Description Generator Prompt Generate SEO-optimized meta descriptions for the following pages: REQUIREMENTS: - Length: 150-160 characters (including spaces) - Include primary keyword near the beginning - Add compelling call-to-action - Highlight unique value proposition - Tone: Professional yet conversational - Avoid: Keyword stuffing, ALL CAPS, excessive punctuation FORMAT: Page Title: [Title] Primary Keyword: [Keyword] USP: [Unique Selling Point] OUTPUT: Provide meta description only, no explanations. Create 3 variations for A/B testing. PAGES TO PROCESS: [Paste your list here]

Batch Process in ChatGPT

For bulk processing:

  1. Copy 10-20 page titles at once
  2. Paste into ChatGPT with the prompt above
  3. Review generated descriptions
  4. Copy to spreadsheet
  5. Repeat until complete

Time per batch: 2-3 minutes for 20 descriptions

Advanced AI Prompts for Different Niches

Different industries need different approaches. Here are specialized prompts:

E-commerce Products

Write meta descriptions for e-commerce products: Structure: - Start with primary keyword - Mention key benefit/feature - Include price or discount if applicable - Add urgency/scarcity: "Limited stock" / "Sale ends soon" - CTA: "Shop now" / "Order today" Example: "Buy [product] online. [Key feature] + [benefit]. Starting at $[price]. Free shipping. Shop now & save 20%!" Tone: Urgent, benefit-focused

Blog Posts/Articles

Create meta descriptions for blog content: Structure: - Hook with question or stat - Promise specific value - Mention number of tips/steps - CTA: "Read more" / "Learn how" Example: "Struggling with [problem]? Discover [number] proven strategies to [solution]. Step-by-step guide with examples. Read now!" Tone: Educational, helpful

SaaS/Software

Generate meta descriptions for SaaS products: Structure: - State the problem you solve - Mention key feature/integration - Include social proof: "Trusted by 10,000+" - CTA: "Start free trial" / "Book demo" Example: "[Software] helps [audience] [solve problem]. [Key feature]. Trusted by 10,000+ teams. Start your free 14-day trial today!" Tone: Professional, trustworthy

Optimization: Making AI Descriptions Click-Worthy

AI generates good descriptions. Optimization makes them great. Here's how:

✅ Do This

  • Keep length 150-160 characters
  • Place keyword in first 60 characters
  • Use action verbs: Discover, Learn, Get, Shop
  • Add numbers: "7 tips," "3 steps," "50% off"
  • Include emotional triggers: "Proven," "Easy," "Fast"
  • Add urgency: "Today," "Now," "Limited time"

❌ Avoid This

  • Don't exceed 160 characters (gets truncated)
  • Don't stuff keywords unnaturally
  • Don't use ALL CAPS or excessive !!!
  • Don't make false promises
  • Don't duplicate descriptions across pages
  • Don't forget mobile (120 char limit)

The Perfect Meta Description Formula

📝 155-Character Template

[Keyword] + [Benefit/Feature] + [Social Proof/Urgency] + [CTA]

"Organic coffee beans (keyword) fresh roasted weekly (benefit). Trusted by 50,000+ customers (proof). Order today & save 20% (CTA)!"

Quality Control Checklist

Before publishing AI-generated descriptions, verify:

  • Character count: 150-160 for desktop, 120 for mobile
  • Keyword placement: Primary keyword in first 60 characters
  • Uniqueness: No duplicate descriptions (use Screaming Frog to check)
  • Readability: Sounds natural when read aloud
  • Accuracy: Matches actual page content
  • CTA included: Clear action for users
  • No truncation: Test in Google Search Console

Common Mistakes to Avoid

⚠️ Mistake #1: Generating descriptions without reviewing. AI makes errors. Always proofread.

⚠️ Mistake #2: Using same prompt for all pages. Different content types need different approaches.

⚠️ Mistake #3: Ignoring character limits. Truncated descriptions hurt CTR.

⚠️ Mistake #4: Forgetting mobile optimization. 60% of searches are mobile.

Tracking Performance & Iterating

After implementing AI-generated meta descriptions, monitor:

  • Click-through rate (CTR) in Google Search Console
  • Average position changes
  • Impressions increase
  • Bounce rate (do descriptions match content?)

A/B Testing Strategy:

  1. Generate 3 variations per page
  2. Implement variation A for 2 weeks
  3. Switch to variation B for 2 weeks
  4. Compare CTR data
  5. Keep winner, test again if needed
